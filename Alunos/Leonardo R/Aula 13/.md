## Conceito escolhido: Suspending Functions

**Timestamp do vídeo:** 2:47

### O que é?

As *suspending functions* são funções especiais do Kotlin que podem ser pausadas durante a execução e retomadas depois, sem travar o restante do programa. Elas são identificadas pela palavra-chave `suspend`.

### Pra que serve?

Elas são muito úteis para tarefas que podem demorar um pouco para terminar, como fazer uma requisição para uma API, acessar um banco de dados ou ler um arquivo. Enquanto a função está esperando essa tarefa ser concluída, o programa pode continuar executando outras atividades normalmente.

### Como é normalmente utilizado?

Uma função `suspend` só pode ser chamada dentro de outra função `suspend` ou de uma *coroutine*. Esse recurso é bastante utilizado no desenvolvimento de aplicativos Android e aplicações que precisam realizar várias tarefas ao mesmo tempo de forma eficiente.

### Exemplo de código

```kotlin
import kotlinx.coroutines.*

suspend fun buscarDados() {
    delay(2000) // Espera 2 segundos sem bloquear o programa
    println("Dados carregados!")
}

fun main() = runBlocking {
    buscarDados()
}
```

Nesse exemplo, a função `buscarDados()` espera dois segundos antes de imprimir a mensagem. Durante essa espera, a execução não bloqueia a aplicação, o que torna o programa mais leve e eficiente.
